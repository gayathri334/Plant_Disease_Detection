# 🌿 Plant Disease Detection

A Machine Learning / Deep Learning based project used to detect plant diseases from leaf images.  
This system helps farmers and agriculture professionals identify plant diseases quickly and accurately using image processing and AI techniques.

---

## 📌 Project Overview

The Plant Disease Detection system analyzes plant leaf images and predicts the disease affecting the plant.  
The project is designed to support smart agriculture and improve crop health monitoring.

---

## 🚀 Features

- Upload plant leaf images
- Detect plant diseases automatically
- Fast and accurate prediction
- User-friendly interface
- Supports multiple plant disease categories
- AI-powered disease classification

## 📸 How It Works

1. Upload a plant leaf image
2. The system preprocesses the image
3. AI model analyzes the leaf
4. Disease prediction is displayed
5. Suggested disease information is provided

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Plant_Disease_Detection.git
```

Go to the project directory:

```bash
cd Plant_Disease_Detection
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
